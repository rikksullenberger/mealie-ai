# Welcome to Mealie AI

!!! info "About Mealie AI"
    **Mealie AI** is an enhanced distribution of [Mealie](https://mealie.io), built on top of the robust open-source recipe manager. This version integrates powerful AI capabilities to streamline your meal planning and recipe management experience.

    **Gratitude:** We are incredibly grateful to the original [Mealie team and contributors](https://github.com/mealie-recipes/mealie/graphs/contributors) for their amazing work in creating the foundation for this project.

## New AI Features
- **AI Recipe Generation**: Generate recipes from simple ideas or names using OpenAI.
- **AI Image Generation**: Automatically generate beautiful images for your recipes using DALL-E.
- **Enhanced Security**: Includes critical patches for `urllib3`, `pip`, `gosu`, and base system libraries (fixing CVE-2025-58188, CVE-2025-66471, and more).
